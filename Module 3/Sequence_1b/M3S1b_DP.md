# Déroulé pédagogique – Module 3 Séquence 1

## PHP avec PDO, CRUD, sessions et emails (3 jours) partie b

### Objectif global

Mettre en place un mini-site web dynamique en PHP utilisant une base de données via PDO, intégrant un CRUD complet, une authentification par session et l’envoi d’emails.

---

## Jour 1 – Accès base de données et CRUD avec PDO

### Contenus

* Rappels SQL utiles au CRUD

* Présentation de PDO

* Connexion à une base de données (DSN, try/catch)

* Requêtes préparées

* Sécurité: injections SQL

* CRUD:
  
  * Create (`INSERT`)
  * Read (`SELECT`)
  * Update (`UPDATE`)
  * Delete (`DELETE`)

### Activités pédagogiques

* Connexion PDO à une base fournie
* Lecture et affichage de données
* Mise en place du CRUD sur une table métier

### Objectif du jour

Être capable d’interagir avec une base de données en PHP via PDO et réaliser un CRUD sécurisé.

---

## Jour 2 – Mini-site web dynamique

### Contenus

* Organisation simple d’un mini-site PHP
* Pages dynamiques (listes, détails)
* Formulaires HTML reliés à la base
* Validation basique des données
* Séparation minimale logique / affichage

### Activités pédagogiques

* Création d’un mini-site web
* Exploitation des données de la base
* Formulaires d’ajout et de modification
* Suppression sécurisée

### Objectif du jour

Construire un mini-site web fonctionnel exploitant une base de données.

---

## Jour 3 – Sessions, authentification et emails

### Contenus

* Principe des sessions PHP

* `session_start()`

* Authentification utilisateur

* Protection des pages

* Déconnexion

* Envoi d’emails:
  
  * `mail()`
  * Notions SMTP
  * Cas d’usage (confirmation, contact)

### Activités pédagogiques

* Mise en place d’un système de login
* Restriction d’accès par session
* Envoi d’un email depuis un formulaire

### Objectif du jour

Sécuriser un mini-site par authentification et intégrer une fonctionnalité d’envoi d’emails.

---

### TP fil rouge – Mini-site PHP

Développement d’un mini-site web comprenant:

* CRUD complet via PDO
* Exploitation d’une base de données métier
* Authentification par session
* Envoi d’emails fonctionnel

---

### Synthèse de la séquence

Mise en œuvre complète d’un backend PHP procédural sécurisé, préparant l’introduction de Symfony.
