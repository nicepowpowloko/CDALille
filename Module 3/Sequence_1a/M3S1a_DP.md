# Déroulé pédagogique – Module 3 Séquence 1 partie a

## Révision SQL et bases du PHP (3 jours)

### Objectif global

Réviser les fondamentaux SQL et acquérir les bases du langage PHP afin de préparer le développement backend et l’accès aux bases de données.

---

## Jour 1 – Révision SQL

### Contenus

* Rappels SGBD et modèle relationnel
* Tables, clés primaires / étrangères
* Requêtes essentielles: `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`
* Fonctions d’agrégation (`COUNT`, `SUM`)
* `GROUP BY`
* Jointures simples (`INNER JOIN`)
* Bonnes pratiques SQL (guillemets, lisibilité)

### Activités pédagogiques

* Exécution de requêtes guidées
* Questions SQL progressives sur une base fournie

### Objectif du jour

Être capable d’écrire et comprendre des requêtes SQL utilisées dans une application backend.

---

## Jour 2 – Bases du PHP

### Contenus

* Rôle du PHP côté serveur
* Structure d’un fichier `.php`
* Variables et types
* Typage dynamique et `null`
* Conditions (`if / else`)
* Tableaux indexés et associatifs
* Parcours avec `foreach`

### Activités pédagogiques

* Scripts PHP simples
* Exercices variables / conditions / tableaux

### Objectif du jour

Écrire un script PHP simple manipulant des données côté serveur.

---

## Jour 3 – PHP procédural

### Contenus

* Fonctions utilisateur
* Paramètres et valeurs de retour
* Dates et heures
* Formulaires (`GET` / `POST`)
* Variables serveur
* Manipulation de fichiers (lecture / écriture)

### Activités pédagogiques

* Création de fonctions simples
* Traitement de formulaires
* Exercices de synthèse PHP

### Objectif du jour

Structurer un script PHP procédural complet et lisible.

---

### Synthèse de la séquence

Révision SQL + bases PHP acquises, préparation à l’accès aux bases de données et aux frameworks backend.
