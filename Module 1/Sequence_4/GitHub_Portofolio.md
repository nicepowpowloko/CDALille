# GitHub, Portfolio & Progression Personnelle

## Objectifs de la journée

- Consolider l’utilisation de Git & GitHub en autonomie
- Manipuler les Pull Requests, reviews et merges sur un dépôt distant
- Créer votre **portfolio développeur** en ligne via GitHub Pages
- Découvrir **Codewars** comme outil d'entraînement régulier

---

## 1) Échauffement : GitHub Skills (activité guidée)

Rendez-vous sur : https://skills.github.com

Choisissez le module :

```
Introduction to GitHub
```

Puis :

- Suivez l’activité **pas à pas**
- Lisez les messages du bot GitHub
- Répondez aux commentaires / tâches demandées
- Validez la Pull Request finale

> Vous pratiquez ici **Pull Request → Review → Merge** exactement comme dans un vrai projet.

---

## 2) Publication de votre site personnel avec GitHub Pages

Restez sur https://skills.github.com et lancez le module :

```
GitHub Pages
```

À la fin de l’activité, votre site sera en ligne à l’adresse :

```
[https://votre-nom-utilisateur.github.io](https://votre-nom-utilisateur.github.io)
```

>  Ce site sera votre **portfolio développeur** : il évoluera avec vous.

---

## 3) Construire votre Book (Portfolio)

Ouvrez votre dépôt `votre-nom-utilisateur.github.io`.

Modifiez le contenu de la page principale (`index.md` ou `index.html`) :

Suggestions de sections :

- **Présentation rapide** : qui vous êtes
- **Ce que vous apprenez actuellement**
- **Projets réalisés en formation**
- **Objectifs professionnels**

### Exemple minimal ( ...vous pouvez/devez faire franchement mieux...) :

```markdown
# Bonjour 
Je me forme actuellement au développement web.

## Compétences en cours d’acquisition
- HTML / CSS / JavaScript
- Git & travail collaboratif sur GitHub

## Mes projets
- (À compléter au fil de la formation)

## Objectifs
Continuer à progresser et réaliser mes premiers projets concrets.
```

>  Vous pouvez personnaliser progressivement : style, images, liens, etc.

---

## 4) Découverte de Codewars (fin de journée)

[https://codewars.com](https://codewars.com)

Créez un compte → choisissez `JavaScript` → commencez par des katas **niveau 8 kyu**.

>  Codewars = entraînement libre → **10 minutes par jour suffisent** pour progresser.
