# **Notice d’introduction – Module d’entraînement JavaScript (M2S5)**

Ce module de trois jours a pour objectif de renforcer votre maîtrise de JavaScript moderne avant l’étude des API sécurisées (M2S6).
Vous allez travailler à la fois la logique du langage, la manipulation du DOM, l’architecture modulaire et l’asynchronisme interne.

Pour vous accompagner, trois ressources complémentaires sont fournies :

---

## **1 Le programme de formation – “JavaScript : Initiation → Consolidation → Expertise”**

Ce document présente l’ensemble des notions prévues dans le parcours CDA, organisées en trois étapes progressives :
bases du langage, consolidation intermédiaire et pratiques avancées.
Cela vous permettra de situer clairement où se placent les trois jours d’entraînement et vers quoi vous devez tendre.

---

## **2 Les sources d’apprentissage – MiniJS + compléments MDN**

MiniJS est votre support principal : un guide clair, synthétique et accessible pour réviser rapidement les fondamentaux.
Certaines notions avancées (DOM, BOM, classes, collections, modules avancés, async…) ne sont pas couvertes dans MiniJS :
des liens MDN officiels vous sont donc fournis pour compléter votre montée en compétences de manière fiable.

Ces deux sources combinées constituent votre bibliothèque de référence pour progresser efficacement.

---

## **3 Les exercices pratiques – 3 jours d’entraînement structuré**

Vous trouverez une série d’exercices progressifs, organisés en ateliers quotidiens :

* Jour 1 : logique, structures de données, objets, JSON
* Jour 2 : DOM avancé, composants, interactions utilisateur
* Jour 3 : modules ES, organisation du code, asynchronisme interne

Chaque journée se clôture par un mini-projet complet permettant d’appliquer concrètement les acquis.

---

## **Comment utiliser ces trois ressources ?**

* **MiniJS** → réviser rapidement, vérifier les bases.
* **MDN** → approfondir une notion ou aller plus loin quand MiniJS ne couvre pas un sujet.
* **Exercices** → mettre en pratique immédiatement, consolider, s’entraîner et préparer la suite du cursus CDA (notamment M2S6 et l’étude des API sécurisées).
