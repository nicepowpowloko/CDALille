# **Déroulé pédagogique – M2S5 – Entraînement JavaScript (3 jours)**

**Objectif global :**
Renforcer la maîtrise de JavaScript moderne (ES6+), structurer du code propre, manipuler efficacement le DOM, travailler avec des modules et comprendre l’asynchronisme interne, afin de préparer M2S6 dédié aux API sécurisées.

---

## **1 – Révisions logiques & manipulation avancée (Jour 1)**

**Durée :** ~1 journée
**Supports :** Programme JavaScript, MiniJS, exercices JS jour 1

### Contenu

* Rappel des bases essentielles : types, variables, opérateurs, fonctions, tableaux, chaînes
* Travail sur la structuration du code et les bonnes pratiques
* Manipulation avancée de données : tableaux complexes, objets, méthodes
* JSON : structure, parse, stringify
* Préparation à la modularisation et au DOM (Jours 2 et 3)

### Activités

1. **Exercice 1 – Validation de données**
2. **Exercice 2 – Tableaux complexes (filtre, tri, total, mapping)**
3. **Exercice 3 – Fonctions avancées (normalisation, transformation)**
4. **Exercice 4 – Objets avec méthodes**
5. **Mini-projet – Gestionnaire JSON** : export, import, modification, réécriture

**Objectif de la journée :** être capable de manipuler des données, structurer proprement son code JS et comprendre les bases nécessaires à la manipulation du DOM.

---

## **2 – DOM, composants & interactions avancées (Jour 2)**

**Durée :** ~1 journée
**Supports :** Exercices jour 2, MiniJS (DOM partiel), compléments MDN

### Contenu

* Sélection et manipulation DOM avancée
* Génération d’éléments, duplication, remplacement
* Gestion d’événements complexes (clics multiples, menu contextuel)
* Création de composants JS réutilisables
* Dynamisation d’un formulaire (lignes multiples)

### Activités

1. **Exercice 5 – Manipulation DOM complète**
2. **Exercice 6 – Composant “Carte” généré en JS**
3. **Exercice 7 – Événements avancés (clic, dblclick, contextmenu) + délégation**
4. **Exercice 8 – Formulaire dynamique**
5. **Mini-projet – Mini-tableur**

   * Cellules éditables
   * Validation & annulation
   * Ajout lignes/colonnes

**Objectif de la journée :** maîtriser le DOM pour concevoir des interfaces dynamiques manipulables en JavaScript pur.

---

## **3 – Modules ES, architecture & asynchronisme interne (Jour 3)**

**Durée :** ~1 journée
**Supports :** Exercices jour 3, MiniJS chapitres Modules & Async, MDN Promises

### Contenu

* Séparation du code en modules ES
* Communication entre modules (imports/exports)
* Architecture front simple : `utils`, `state`, `ui`, `main`
* Événements personnalisés entre modules
* Promises, `setTimeout`, erreurs, simulation d’asynchronisme sans API
* Méthodes avancées : `Promise.race`, `Promise.all`

### Activités

1. **Exercice 9 – Construction d’une application en modules**
2. **Exercice 10 – Système d’événements personnalisés**
3. **Exercice 11 – Simulation de chargement asynchrone (sans API)**
4. **Exercice 12 – Promises avancées**
5. **Mini-projet – Application modulaire complète**

   * gestion du state
   * gestion du DOM
   * tri, filtrage, persistance localStorage
   * orchestration avec `main.js`

**Objectif de la journée :** comprendre comment structurer une application JavaScript complète, propre et maintenable, préparant M2S6 sur les API sécurisées.
