# **Déroulé pédagogique – Manipulation de JSON en JavaScript (1 jour)**

## **Objectif global**

Comprendre et manipuler le format JSON pour préparer l’exploitation de données issues d’API REST : conversion JSON ↔ objets JS, filtrage, transformation et intégration dans des systèmes JavaScript modernes.

---

## **Contenus de la journée**

### **1 – Comprendre le JSON**

* Structure d’un JSON : objets, tableaux, valeurs.
* Différence entre JSON (texte) et objets JS.
* Parcours d’un JSON représentant des personnages/animaux (exemples du cours).

### **2 – JSON ↔ Objets JavaScript**

* `JSON.parse()` pour convertir un texte JSON en objets JS.
* `JSON.stringify()` pour générer un JSON à partir d’un objet JS.
* Lecture de propriétés, accès aux objets imbriqués, bonnes pratiques.

### **3 – Manipulation de tableaux JSON**

* Parcours avec `forEach`.
* Filtrage (`filter`), transformation (`map`), recherche (`find`).
* Extraction de valeurs (noms, races, types…).
* Approche progressive à partir des exercices (Gimli, Legolas, Frodo…).

### **4 – Préparation à l’usage d’API REST**

* Le rôle de `response.json()` dans `fetch()`.
* Traitement d’un tableau renvoyé par une API.
* Gestion d’erreurs simples.
* Introduction : transformer un JSON en instances de classes (lien POO).

---

## **Activités pédagogiques**

* **Exercice 1** – Parser un JSON, accéder aux propriétés.
* **Exercice 2** – Convertir un objet JS en texte JSON (validation du type).
* **Exercice 3** – Manipuler un tableau JSON (affichage, filtrage, mapping).
* **Exercice 4** – Fonction de filtrage par race.
* **Exercice 5** – Transformation en phrases (`map`).
* **Exercice 6** – JSON → instances de classes (`Personnage`, `Elfe`, etc.).
* **Exercice 7** – Traitement d’un tableau renvoyé par une API.

**Objectif de la journée :** être capable de lire, transformer et exploiter des données JSON pour l’intégrer dans un code JavaScript propre et évolutif, première étape avant les API REST.
